02_kalman_dynamic_kgain
Author: Makoto Kawamura

本プロジェクトは、Tang Nano 9Kを用いたカルマンフィルタ実装シリーズの第2段階です。
前回のゲイン固定方式から進化し、除算器（Divider）を実装することでカルマンゲイン K を動的に演算する構成となっています。

AI-Assisted Development:
本プロジェクトの設計、RTLコードの最適化、およびドキュメント作成には、AIモデル Gemini をフルに活用しています。

プロジェクト概要
カルマンフィルタの本質である「予測」と「更新」のサイクルをフル実装しています。特に、誤差共分散 P と観測ノイズ R から最適なカルマンゲイン K = P / (P + R) を算出するための除算回路を統合した点が最大の特徴です。

性能諸元（理論値）
本実装における計算処理の性能は以下の通りです。

動作周波数: 27.0 MHz (Clock Period: 37.04 ns)
演算サイクル数: 約 40 サイクル
予測ステップ等: 8 サイクル
除算処理: 32 サイクル（逐次比較型除算器による演算）
演算処理時間: 約 1,481.6 ns (演算開始から推定値確定まで)
データ更新間隔: 200 ms (5.4Mサイクル / 5Hz設定)

特徴

逐次比較型（Non-restoring division類似）32bit 除算器による動的ゲイン演算
誤差共分散 P のリアルタイム更新による適応的なフィルタリング
Q16.16 固定小数点演算による高精度な推論
UARTによる観測値および推定値のリアルタイム出力

開発環境

FPGA: Tang Nano 9K (GW1NR-LV9QN88PC6/I5)
IDE: GOWIN FPGA Designer (Gowin EDA)
AI Collaborator: Google Gemini

使用方法
Gowin EDAで本ディレクトリ内のプロジェクトファイルを開き、ビルド後に実機へ書き込んでください。固定ゲイン版と比較して、起動直後の収束の速さや、ノイズに対する適応能力の違いを確認できます。
