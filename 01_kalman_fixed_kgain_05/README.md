Author: Makoto Kawamura

本プロジェクトは、Tang Nano 9Kを用いたカルマンフィルタ実装シリーズの第1段階です。
最も基本的な構成として、カルマンゲイン K を 0.5 に固定した実装を行っています。

AI-Assisted Development:
本プロジェクトの設計、RTLコードの最適化、およびドキュメント作成には、AIモデル Gemini をフルに活用しています。

プロジェクト概要
カルマンフィルタの演算において最も負荷のかかる除算を排除し、定数ゲイン（K=0.5）を用いることで、ビットシフト演算のみによる軽量なフィルタリングを実現しています。FPGAでの信号処理の基礎確認を目的としています。

性能諸元（理論値）
本実装における計算処理の性能は以下の通りです。

動作周波数: 27.0 MHz (Clock Period: 37.04 ns)
演算サイクル数: 5 サイクル (Step 1からStep 5までの確定待ち)
演算処理時間: 185.2 ns (演算開始から推定値確定まで)
データ更新間隔: 200 ms (5.4Mサイクル / 5Hz設定)

特徴

カルマンゲイン K=0.5 固定による高速・低リソース実装
Q16.16 固定小数点演算の採用
疑似的な正規分布ノイズ生成器の搭載
UARTによる観測値および推定値のリアルタイム出力
LEDによる動作ステータスの可視化

開発環境

FPGA: Tang Nano 9K (GW1NR-LV9QN88PC6/I5)

IDE: GOWIN FPGA Designer (Gowin EDA)

AI Collaborator: Google Gemini

使用方法
Gowin EDAで本ディレクトリ内のプロジェクトファイルを開き、論理合成および配置配線を行った後、実機に書き込んでください。PC上のシリアルモニタ（115200bps）でフィルタの効果を確認できます。
